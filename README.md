# 📊 Furniture Sales Analysis Dashboard

An interactive Microsoft Excel dashboard built to analyze furniture sales performance across different regions, customer segments, products, and time periods. This project demonstrates my ability to clean data, build Pivot Tables, create KPIs, and design interactive dashboards for business decision-making.

---

## 📌 Project Overview

The objective of this project was to transform raw furniture sales data into meaningful business insights through Excel data analysis and visualization.

The dashboard enables stakeholders to monitor key business metrics, identify sales trends, evaluate product performance, and understand regional profitability.

---

## 🎯 Business Questions Answered

This dashboard answers questions such as:

- What is the total sales revenue?
- What is the total profit generated?
- How many orders were completed?
- Which regions generate the highest sales?
- Which product categories perform best?
- What are the monthly sales trends?
- Which customer segments contribute the most revenue?
- Which sub-categories are the most profitable?

---

The dashboard includes:
